# Meto Parts Library

素材管理系统 — 支持 7 大类素材（种草/直播/1V1/匹配/真实展示/剧情/原生）的集中管理与快速检索。

## 功能
- 素材思维导图结构：7 大类 × 钩子类型 × (前3s/中部/CTA)
- YT 素材管理（iOS / Android）
- 地区分类与筛选
- 素材库导入/导出
- 主题切换
- 批量新增
- 本地存储（localStorage）

## 部署
部署到 Cloudflare Pages 或其他静态托管平台。

## 本地开发
```bash
node server.js
```
访问 `http://localhost:3000/meto-parts-library.html`
